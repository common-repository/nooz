Press Release Options

The Press Release tab contains options to modify the behavior of a press release page.

  • URL

    By default press releases will be published with the following permalink /news/press-releases/{{slug}} structure. Take care when changing this option as it will affect page links. Your cache may need to be cleared when changing this option.

  • Featured Image

    This is the default featured image that will appear if one is not added for a press release.

  • Location

    The location precedes the press release and helps to orient the reader (e.g. San Francisco, CA).

  • Date Format

    The date follows the location. Leave this blank to use the default date format as set in WordPress. Learn more about formatting dates.

  • Dateline Format

    The dateline precedes the press release content and is usually a combination of the location and date followed by a long-dash.

  • Boilerplate

    The boilerplate is a few sentences at the end of your press release that describes your organization. This should be used consistently on press materials and written strategically, to properly reflect your organization. Using HTML in this field is allowed. This option helps keep your press release content separate.

  • Ending

    The ending mark signifies the absolute end of the press release (e.g. ###, END, XXX, -30-). Removing this value will disable this option.

  • Media Contact

    The media contact is the person or organization that can be contacted for more information about the press release.
